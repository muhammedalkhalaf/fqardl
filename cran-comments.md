## Package Description

`fqardl` implements advanced ARDL methodologies for cointegration analysis:

1. **FQARDL** - Fourier Quantile ARDL
2. **FNARDL** - Fourier Nonlinear ARDL (Shin et al., 2014)
3. **MTNARDL** - Multi-Threshold NARDL
4. **Fourier Unit Root Tests** - ADF and KPSS with Fourier terms

Originally ported from Stata/Python implementations by Dr. Merwan Roudane.
